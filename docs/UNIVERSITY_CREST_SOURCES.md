# University GAA Club Crest Sources

This document lists likely sources for finding crests/logos for each university GAA club.
After downloading images, place them in a folder and run the bulk upload script.

## How to Use

1. Click the links below to find logos
2. Right-click and save any good quality crests you find
3. Name the file using the format: `club-name.png` (e.g., `dcu-dochas-eireann.png`)
4. Place all images in: `public/crests/universities/`
5. Run: `npx tsx scripts/bulk-upload-university-crests.ts`

## Tips for Finding Crests

1. **Twitter/X Profile Pictures** - Often the best source for GAA-specific crests
2. **Instagram Highlights** - Sometimes have logo in cover images
3. **University Sports Pages** - Search "[University] GAA club"
4. **GAA HE Website** - https://www.gaa.ie/highereducation may have team logos
5. **Facebook Pages** - Many clubs have active FB pages with logos

## Image Requirements

- **Format**: PNG preferred (transparent background), JPG acceptable
- **Size**: Minimum 200x200px, ideally 400x400px or larger
- **Quality**: Clear, not pixelated
- **Naming**: Use kebab-case matching club name (e.g., `dcu-dochas-eireann.png`)
