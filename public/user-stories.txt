Manual Testing User Journeys

  Journey 1: New User Registration & Event Discovery

  Persona: Sarah, a GAA player living in Berlin looking for local events

  1. Landing Page
    - Visit homepage, verify map loads with club markers
    - Check responsive design on mobile/desktop
    - Click on a club marker, verify popup shows club info
  2. User Registration
    - Click "Sign Up" button
    - Fill registration form with valid details
    - Select a club from dropdown (test club selector loading)
    - Submit form and verify success message
    - Check that account status shows "PENDING"
  3. Event Discovery
    - Navigate to Events page
    - Verify events display with club logos in top-left corners
    - Test event filters (type, country, month)
    - Click on an event to view details
    - Verify structured data appears in page source

  Expected Outcome: User successfully registers and can browse events
